虽不是最简单的教程，但绝对是最正确的教程

一. 背景

其实早就有想搭建博客这个想法，但是因为觉得自己底子都还不够，又怎能写博去教人，自己写写笔记就好啦。于是每每想起搭博，就用这个借口来搪塞自己。后来逐渐的越来越懒，笔记也就慢慢的越做越少，虽想整个博客嫌麻烦，就一直耽搁着。最后趁着这次小鹿锅带头搭建个人博客，索性就咬咬牙，克服下自己的懒惰！
